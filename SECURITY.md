# Security

This is a private Tailrocks Pte. Ltd. repository.

Do not disclose vulnerabilities, secrets, private research, unpublished product
plans, repository contents, screenshots, logs, or access details publicly.

Report security concerns directly to the Tailrocks repository owner or through
the private channel provided to you by Tailrocks Pte. Ltd.

If you received access by mistake, do not clone, copy, forward, or share the
repository. Notify Tailrocks Pte. Ltd. and delete any local copies.
